# Kỷ Nguyên Phân Tích Bóng Đá

Website tĩnh sao chép giao diện từ trang Wix về phân tích bóng đá.

## Mô tả

Trang web này bao gồm các phần:
- Tiêu đề và banner
- Danh sách các trận đấu với phân tích
- Phần phân tích chuyên sâu
- Lời kêu gọi tham gia cộng đồng
- Footer với liên kết xã hội

## Cách chạy

Mở file `index.html` trong trình duyệt web để xem trang web.

## Lưu ý

- Hình ảnh banner là placeholder. Thay thế `banner.png` bằng hình ảnh thực tế.
- Liên kết "Xem Phân Tích" hiện tại chỉ hiển thị alert. Có thể mở rộng để hiển thị nội dung thực.
